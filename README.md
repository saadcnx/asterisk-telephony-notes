# 📞 Asterisk PBX — Learning Notes & Configuration Reference

Personal study notes and configuration examples from a full walkthrough of the **"Introducing Asterisk" tutorial series**, covering the setup, configuration, and administration of the **Asterisk open-source PBX / telephony platform**.

This repository documents my hands-on learning journey — from installation to advanced call routing — organized as a structured reference that others (and future me) can use.

---

## 🎯 Why This Repo Exists

I'm learning Asterisk (open-source telephony / VoIP PBX software) to build practical, job-ready skills in:

- Linux-based telephony systems
- VoIP protocols (SIP / IAX)
- Dialplan scripting
- Call routing, IVR, voicemail, and queue management

Each topic below has its own notes file with **key concepts, configuration snippets, and lessons learned**.
